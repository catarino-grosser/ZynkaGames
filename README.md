# Zynka RPG - Protótipo

Protótipo de plataforma de RPG interativo em HTML, CSS e JavaScript.

## O que já tem

- Tela inicial com várias aventuras.
- Criação de personagem.
- Classes: Guerreiro, Mago e Ladino.
- Sistema de escolhas.
- Batalha simples por turno.
- Vida, ataque, ouro e inventário.
- Salvamento automático no navegador com localStorage.
- Estrutura preparada para trocar localStorage por Firebase depois.

## Como usar no SPCK Editor

1. Crie uma pasta nova no SPCK.
2. Coloque os arquivos `index.html`, `style.css` e `script.js` dentro dela.
3. Abra o `index.html` para testar.
4. Envie para o GitHub.
5. Conecte o repositório no Netlify.
